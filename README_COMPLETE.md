## ⚙️ 配置说明

### 服务器配置

**文件：** `server.py`

```python
# 上传文件夹
UPLOAD_FOLDER = './received_files'

# 允许的文件类型
ALLOWED_EXTENSIONS = {
    'png', 'jpg', 'jpeg', 'gif', 'bmp', 'webp', 'svg',
    'pdf', 'doc', 'docx', 'xls', 'xlsx', 'txt',
    'zip', 'rar', 'mp4', 'mp3', 'avi', 'mov'
}

# 最大文件大小（100MB）
MAX_FILE_SIZE = 100 * 1024 * 1024

# 服务端口
port = 5000
```

### 修改端口

编辑 `server.py` 最后一行：
```python
app.run(host='0.0.0.0', port=5000, debug=False)
# 改为其他端口，如：port=8080
```

## ❓ 常见问题

### Q1: 手机无法连接？

**解决方法：**
1. 确保手机和电脑在同一WiFi
2. 检查防火墙是否阻止5000端口
3. 尝试关闭杀毒软件
4. 使用正确的IP地址（非127.0.0.1）

---

### Q2: 上传失败？

**可能原因：**
- 文件超过100MB
- 文件格式不支持
- 网络不稳定

**解决方法：**
- 检查文件大小
- 确认文件类型在允许列表中
- 重试上传

---

### Q3: 自动刷新不工作？

**检查步骤：**
1. 打开浏览器开发者工具（F12）
2. 查看Console是否有错误
3. 检查Network面板 `/api/history` 请求
4. 刷新页面（Ctrl+F5）

---

### Q4: 如何清空历史记录？

**方法1：** 点击"清空全部"按钮  
**方法2：** 删除 `received_files/` 文件夹中的所有文件  
**方法3：** 重启服务器（内存中的记录会清空）

---

### Q5: 可以自定义主题色吗？

**可以！** 编辑 `web_templates.py`：

```css
/* 查找渐变色定义 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 修改为你喜欢的颜色 */
background: linear-gradient(135deg, #ff6b6b 0%, #ee5a6f 100%);
```

## 📊 API接口文档

### 获取历史记录

```
GET /api/history
```

**响应：**
```json
{
    "success": true,
    "history": [...],
    "count": 5
}
```

---

### 上传文件

```
POST /api/upload/file
Content-Type: multipart/form-data
```

**参数：**
- `file`: 文件对象

---

### 发送文本

```
POST /api/upload/text
Content-Type: application/json
```

**参数：**
```json
{
    "text": "要发送的文本"
}
```

---

### 下载文件

```
GET /api/download/<filename>
```

**说明：**
- 图片文件：浏览器预览
- 其他文件：强制下载

---

### 删除记录

```
DELETE /api/history/delete/<filename>
```

---

### 清空历史

```
POST /api/history/clear
```

## 🎯 最佳实践

### 使用建议

1. **保持服务器运行**
   - 不要关闭服务器窗口
   - 最小化到任务栏即可

2. **定期清理文件**
   - 避免磁盘空间不足
   - 删除不再需要的文件

3. **网络安全**
   - 仅在可信网络使用
   - 不要在公共WiFi使用

4. **大文件传输**
   - 建议使用有线网络
   - 避免同时传输多个大文件
