## 🔑 核心模块详解

### 1. 设备自动检测

**位置：** `server.py`

**工作原理：**
```python
@app.route('/')
def index():
    # 分析User-Agent字符串
    user_agent = request.headers.get('User-Agent', '').lower()
    
    # 匹配移动设备关键词
    is_mobile = any(keyword in user_agent for keyword in [
        'mobile', 'android', 'iphone', 'ipad'
    ])
    
    # 返回对应模板
    if is_mobile:
        html_template = get_mobile_html_template()
    else:
        html_template = get_desktop_html_template()
    
    return render_template_string(html_template, ...)
```

**检测流程：**
```
用户访问主页 → 获取User-Agent头 → 匹配移动设备关键词 → 返回对应UI模板
├─ 移动端 → 垂直流式布局
└─ 桌面端 → 左右分栏布局
```

### 2. 实时自动刷新机制

const REFRESH_INTERVAL = 100; // 0.1秒

// 启动自动刷新
function startAutoRefresh() {
    autoRefreshInterval = setInterval(() => {
        if (isAutoRefreshEnabled) {
            loadHistory();
        }

// AJAX加载历史记录

    const response = await fetch('/api/history');

    if (data.success) {
        updateHistoryList(data.history);

}

### 3. 文件处理流程

**上传流程：**
```
用户选择文件/拖拽 → POST /api/upload/file → 验证扩展名 → 验证文件大小（≤100MB）
→ 添加时间戳前缀 → 保存到 received_files/ → Magic Number验证 → SVG安全检查
→ 添加到内存历史记录 → 返回成功响应
```

**关键代码：**
```python
@app.route('/api/upload/file', methods=['POST'])
def upload_file():
    file = request.files['file']
    
    # 验证扩展名
    if not allowed_file(file.filename):
        return jsonify({'success': False}), 400
    
    # 生成唯一文件名
    timestamp = datetime.datetime.now().strftime('%Y%m%d_%H%M%S_')
    filename = timestamp + secure_filename(file.filename)
    filepath = os.path.join(UPLOAD_FOLDER, filename)
    
    # 保存文件
    file.save(filepath)
    
    # Magic Number验证
    is_valid, detected_mime = validate_file_magic_number(filepath)
    if not is_valid:
        os.remove(filepath)
        return jsonify({'success': False}), 400
    
    # SVG安全检查
    if filename.lower().endswith('.svg'):
        sanitize_svg(filepath)
    
    # 记录历史
    transfer_history.append({...})
    
    return jsonify({'success': True, 'filename': filename})
```

### 4. 文本传输机制

**保存方式：** 文本内容保存为 `.txt` 文件  
**文件格式：** `text_YYYYMMDD_HHMMSS.txt`

**实现代码：**
```python
@app.route('/api/upload/text', methods=['POST'])
def upload_text():
    data = request.get_json()
    text = data['text']
    
    # 检测是否为链接
    is_link = text.startswith(('http://', 'https://', 'www.'))
    
    # 保存为文本文件
    timestamp = datetime.datetime.now().strftime('%Y%m%d_%H%M%S')
    filename = f"text_{timestamp}.txt"
    filepath = os.path.join(UPLOAD_FOLDER, filename)
    
    with open(filepath, 'w', encoding='utf-8') as f:
        f.write(text)
    
    # 记录历史
    transfer_history.append({
        'type': 'text',
        'content': text,
        'is_link': is_link
    })
    
    return jsonify({'success': True})
```

## 📡 API接口参考

### 基础信息

**Base URL:** `http://192.168.x.x:5000`  
**协议:** HTTP/HTTPS  
**数据格式:** JSON  
**CORS:** 已启用（允许跨域）

---

### 接口列表

#### 1. 获取历史记录
**接口：** `GET /api/history`  
**说明：** 获取最近20条传输记录（用于自动刷新）

**响应示例：**
```json
{
    "success": true,
    "history": [
        {
            "type": "image",
            "type_name": "🖼️ 图片",
            "name": "photo.jpg",
            "filename": "20260427_103045_photo.jpg",
            "time": "2026-04-27 10:30:45",
            "content": "",
            "is_link": false
        }
    ],
    "count": 1
}
```

---

#### 2. 上传文件
**接口：** `POST /api/upload/file`  
**Content-Type:** `multipart/form-data`

**参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| file | File | ✅ | 要上传的文件 |

**成功响应：**
```json
{
    "success": true,
    "filename": "20260427_103045_photo.jpg"
}
```

**错误码：** 400（无文件/不支持类型）、413（文件过大>100MB）

---

#### 3. 发送文本
**接口：** `POST /api/upload/text`  
**Content-Type:** `application/json`

**参数：**
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| text | String | ✅ | 要发送的文本内容 |

**成功响应：**
```json
{
    "success": true
}
```

---

#### 4. 下载文件
**接口：** `GET /api/download/<filename>`

**说明：** 
- 图片文件：浏览器预览（inline）
- 其他文件：强制下载（attachment）

**支持的图片格式（预览）：** PNG, JPG, JPEG, GIF, BMP, WebP, SVG  
**支持的其他格式（下载）：** PDF, DOC, DOCX, XLS, XLSX, TXT, ZIP, RAR, MP4, MP3, AVI, MOV

---

#### 5. 删除记录
**接口：** `DELETE /api/history/delete/<filename>`  
**说明：** 删除单个历史记录及对应文件

**成功响应：**
```json
{
    "success": true,
    "message": "已删除 1 条记录"
}
```

---

#### 6. 清空历史
**接口：** `POST /api/history/clear`  
**说明：** 清空所有历史记录和文件（⚠️ 不可恢复）

**成功响应：**
```json
{
    "success": true,
    "message": "已清空 5 条记录"
}
```

---

#### 7. 列出文件
**接口：** `GET /api/files`  
**说明：** 列出所有接收的文件（不包含元数据）

---

#### 8. 设备信息
**接口：** `GET /api/devices`  
**说明：** 返回服务器信息

**响应：**
```json
{
    "success": true,
    "ip": "192.168.0.102",
    "port": 5000,
    "url": "http://192.168.0.102:5000",
    "device_name": "Windows PC",
    "status": "online"
}
```

---

### 响应状态码

| 状态码 | 说明 | 常见场景 |
|--------|------|----------|
| 200 | 成功 | 正常响应 |
| 400 | 请求错误 | 参数缺失/格式错误 |
| 404 | 未找到 | 文件/记录不存在 |
| 413 | 文件过大 | 超过100MB限制 |
| 500 | 服务器错误 | 内部异常 |

## 🔐 安全机制

### 1. HTTPS加密传输（可选）

**证书生成：**
```bash
python generate_cert.py
```

**生成文件：**
- `cert.pem` - SSL证书
- `key.pem` - 私钥

**安装证书（消除浏览器警告）：**

**Windows:**
```bash
certutil -addstore -f "ROOT" cert.pem
```

**Android:** 设置 → 安全 → 从存储设备安装证书  
**iOS:** 设置 → 通用 → 关于本机 → 证书信任设置

---

### 2. 文件类型验证

#### 扩展名检查

**允许的扩展名：**
```python
ALLOWED_EXTENSIONS = {
    # 图片
    'png', 'jpg', 'jpeg', 'gif', 'bmp', 'webp', 'svg',
    # 文档
    'pdf', 'doc', 'docx', 'xls', 'xlsx', 'txt',
    # 压缩
    'zip', 'rar',
    # 媒体
    'mp4', 'mp3', 'avi', 'mov'
}
```

#### Magic Number验证

**原理：** 读取文件头部字节，识别真实文件类型

**防护效果：**
- ✅ 防止扩展名欺骗（malware.exe → malware.jpg）
- ✅ 识别伪装文件
- ✅ 阻止恶意文件上传

---

### 3. SVG XSS防护

**风险：** SVG文件可嵌入JavaScript脚本，导致XSS攻击

**防护措施：**
- 移除 `<script>` 标签
- 移除事件处理器（onclick, onerror, onload等）
- 移除 `javascript:` 协议
- 过滤其他潜在XSS向量

---

### 4. 速率限制

**目的：** 防止暴力攻击和DoS攻击

**限制策略：**
- **默认限制：** 1000次/天，200次/小时
- **历史查询：** 600次/分钟（支持0.1秒刷新）
- **文件上传：** 继承默认限制
- **文本发送：** 继承默认限制

---

### 5. 文件大小限制

**最大上传：** 100MB

**超限响应：**
```json
{
    "success": false,
    "message": "文件过大"
}
```

**HTTP状态码：** 413 Payload Too Large

---

### 6. CORS跨域支持

**配置：**
```python
from flask_cors import CORS
CORS(app)
```

**作用：** 允许不同域名的网页调用API（如Android App）
