## 🚀 本次更新亮点

### 1. 📐 左右分栏布局（Desktop First）

**改进前：** 垂直堆叠，需要滚动查看  
**改进后：** 
- **左侧面板**：二维码、服务器地址、文件上传区
- **右侧面板**：文本发送、传输历史
- **优势**：一屏可见所有功能，无需滚动

**响应式设计：**
- ✅ 桌面端（≥1024px）：左右分栏
- ✅ 移动端（<1024px）：自动切换为垂直布局

---

### 2. 🗑️ 历史记录管理功能

#### 单个删除
- 每条历史记录右侧添加"🗑️"删除按钮
- 点击后弹出确认对话框
- 删除时带有滑出动画效果
- 同时删除对应文件，释放空间

#### 一键清空
- 历史区域顶部添加"🗑️ 清空全部"按钮
- 红色渐变设计，醒目易识别
- 点击后二次确认，防止误操作
- 清空所有记录和文件

---

### 3. 📍 服务器地址优化

**改进前：** `http://192.168.43.34:5000`  
**改进后：** `http://192.168.43.34:5000 (本机地址)`

**优势：**
- 更直观地理解这是当前电脑的地址
- 减少用户困惑
- 提升可用性

## 📋 新增 API 接口

### DELETE /api/history/delete/<filename>

**功能：** 删除单个历史记录及对应文件

**请求示例：**
```javascript
fetch('/api/history/delete/20260427_123456_photo.jpg', {
    method: 'DELETE'
})
```

**响应：**
```json
{
    "success": true,
    "message": "已删除 1 条记录"
}
```

---

### POST /api/history/clear

**功能：** 清空所有历史记录和文件

**请求示例：**
```javascript
fetch('/api/history/clear', {
    method: 'POST'
})
```

**响应：**
```json
{
    "success": true,
    "message": "已清空 15 条记录"
}
```
