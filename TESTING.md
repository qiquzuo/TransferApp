# 🧪 快速测试指南

## 第一步：启动Windows服务器

```bash
# 在项目根目录执行
cd c:\Users\Lenovo\IdeaProjects\TransferApp

# 方法1：双击运行
start_server.bat

# 方法2：命令行运行
python server.py
```

启动成功后会看到类似输出：
```
============================================================
🚀 局域网文件传输服务器启动成功！
============================================================
📍 服务器地址: http://192.168.1.100:5000
📱 手机浏览器访问上述地址即可上传文件
📁 文件保存目录: C:\Users\Lenovo\IdeaProjects\TransferApp\received_files
============================================================
```

**记下显示的IP地址！**（例如：192.168.1.100）

## 第二步：测试Web界面（无需安装App）

### 在手机上测试：

1. **确保手机和电脑在同一WiFi**

2. **打开手机浏览器**
   - Chrome、Safari或其他浏览器均可

3. **输入服务器地址**
   ```
   http://192.168.1.100:5000
   ```
   （将IP替换为你看到的实际IP）

4. **测试功能**
   - ✅ 页面应该显示紫色渐变背景和二维码
   - ✅ 点击"拖拽文件到此处"可选择文件上传
   - ✅ 在文本框输入文字，点击"发送文本"
   - ✅ 点击"📋 粘贴"可粘贴剪贴板内容

5. **查看接收的文件**
   - 打开电脑的 `received_files` 文件夹
   - 应该能看到从手机发送的文件

### 在电脑浏览器测试：

1. 打开浏览器访问：`http://localhost:5000`
2. 测试上传文件和发送文本功能

## 第三步：测试Android App（可选）

### 使用Android Studio编译：

1. **打开项目**
   ```
   File → Open → 选择 android-app 文件夹
   ```

2. **等待Gradle同步**
   - 首次同步可能需要几分钟
   - 确保网络畅通

3. **修改服务器IP**
   - 打开 `MainActivity.kt`
   - 找到第37行：`private var serverIp = "192.168.1.100"`
   - 改为你的实际IP地址

4. **运行App**
   - 连接Android手机（开启USB调试）
   - 点击Run按钮或按 Shift+F10

5. **测试功能**
   - 点击"🔗 连接服务器"
   - 状态变为"🟢 已连接"表示成功
   - 测试发送文本、图片、文件
   - 测试从其他App分享内容

### 生成APK安装：

1. **构建APK**
   ```
   Build → Build Bundle(s) / APK(s) → Build APK(s)
   ```

2. **找到APK文件**
   ```
   android-app/app/build/outputs/apk/debug/app-debug.apk
   ```

3. **传输到手机安装**
   - 通过数据线、QQ、微信等方式发送到手机
   - 在手机上安装APK

## 第五步：常见问题排查

### 问题1：无法连接服务器

**症状：** 手机浏览器显示"无法访问此网站"

**解决方案：**
1. 确认手机和电脑在同一WiFi
2. 检查Windows防火墙设置
3. 尝试关闭防火墙后重试
4. 确认IP地址正确

**检查防火墙：**
```
控制面板 → Windows Defender 防火墙 → 允许应用通过防火墙
找到 Python → 勾选"专用"和"公用"
```

### 问题2：IP地址变化

**症状：** 之前能用，突然不能用了

**解决方案：**
1. 重新运行 `start_server.bat`
2. 查看新的IP地址
3. 更新手机上的IP地址

**获取固定IP（可选）：**
在路由器中为电脑设置静态IP分配

### 问题3：上传失败

**症状：** 显示"上传失败"或"不支持的文件类型"

**解决方案：**
1. 检查文件大小是否超过100MB
2. 检查文件类型是否在白名单中
3. 查看服务器控制台错误信息

**添加文件类型：**
编辑 `server.py` 第15行：
```python
ALLOWED_EXTENSIONS = {'png', 'jpg', 'jpeg', ...}
# 添加你需要的扩展名
```

### 问题4：Android Studio编译错误

**症状：** Gradle同步失败

**解决方案：**
1. 检查网络连接（需要访问Google仓库）
2. 使用代理或镜像源
3. 更新Android Studio到最新版本
4. 清除缓存：File → Invalidate Caches / Restart
