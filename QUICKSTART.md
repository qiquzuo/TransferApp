# ⚡ 5分钟快速上手指南

## 🎯 目标
在5分钟内实现手机和电脑之间的文件传输

---

## 第一步：启动Windows服务器（1分钟）

### 1.1 打开项目文件夹
```
C:\Users\Lenovo\IdeaProjects\TransferApp
```

### 1.2 双击运行
```
start_server.bat
```

### 1.3 查看输出
你会看到类似这样的信息：
```
============================================================
🚀 局域网文件传输服务器启动成功！
============================================================
📍 服务器地址: http://192.168.1.100:5000
📱 手机浏览器访问上述地址即可上传文件
📁 文件保存目录: C:\Users\Lenovo\IdeaProjects\TransferApp\received_files
============================================================
```

**🔑 关键信息：** 记下IP地址 `192.168.1.100`（你的可能不同）

---

## 第二步：从手机发送文件（2分钟）

### 方法A：使用手机浏览器（最简单，无需安装App）

#### 2A.1 打开手机浏览器
- Chrome、Safari、Edge等任意浏览器

#### 2A.2 输入地址
```
http://192.168.1.100:5000
```
⚠️ **重要：** 将 `192.168.1.100` 替换为你看到的实际IP

#### 2A.3 上传文件
1. 看到紫色渐变页面 = ✅ 连接成功
2. 点击"拖拽文件到此处或点击选择"
3. 选择照片或文件
4. 等待上传完成

#### 2A.4 查看结果
在电脑的 `received_files` 文件夹中查看接收的文件

---

### 方法B：使用Android App（功能更丰富）

#### 2B.1 编译App（首次使用）
```bash
# 用Android Studio打开 android-app 文件夹
# 等待Gradle同步完成
# 点击Run按钮安装到手机
```

#### 2B.2 连接服务器
1. 打开App
2. 输入IP地址：`192.168.1.100`
3. 点击"🔗 连接服务器"
4. 看到"🟢 已连接" = ✅ 成功

#### 2B.3 发送内容

**发送文本：**
1. 点击"📝 文本"
2. 输入文字
3. 点击"发送"

**发送图片：**
1. 点击"🖼️ 图片"
2. 从相册选择
3. 自动上传

**发送文件：**
1. 点击"📁 文件"
2. 选择文件
3. 等待上传

**从剪贴板粘贴：**
1. 复制任意文本
2. 点击"📋 粘贴"
3. 自动发送

## 🐛 遇到问题？

### 问题1：浏览器无法访问

**症状：** 显示"无法访问此网站"或"连接超时"

**解决步骤：**
1. ✅ 确认手机和电脑在同一WiFi
2. ✅ 确认IP地址正确（重新运行 `start_server.bat` 查看）
3. ✅ 尝试关闭Windows防火墙
4. ✅ 尝试用手机流量热点（创建热点让电脑连接）

**快速测试：**
在电脑浏览器访问 `http://localhost:5000`
- 如果能访问 = 服务器正常，是网络问题
- 如果不能访问 = 服务器未启动

---

### 问题2：连接失败

**症状：** App显示"🔴 连接失败"

**解决步骤：**
```bash
# 1. 检查服务器是否运行
# 看命令行窗口是否有错误信息

# 2. 测试网络连接
# 在手机浏览器输入服务器地址，看能否访问

# 3. 检查防火墙
控制面板 → Windows Defender 防火墙 → 允许应用通过防火墙
找到 Python → 勾选"专用"和"公用"
```

---

### 问题3：上传失败

**症状：** 显示"上传失败"或"不支持的文件类型"

**解决步骤：**
1. 检查文件大小 < 100MB
2. 检查文件类型是否在支持列表中
3. 查看服务器控制台的错误信息

**支持的文件类型：**
- 图片：png, jpg, jpeg, gif, bmp
- 文档：pdf, doc, docx, xls, xlsx, txt
- 压缩：zip, rar
- 媒体：mp4, mp3

## 💡 实用技巧

### 技巧1：固定IP地址
避免每次IP变化，在路由器中设置静态IP分配

### 技巧2：创建桌面快捷方式
```
右键 start_server.bat → 发送到 → 桌面快捷方式
```

### 技巧3：开机自启动
```
1. Win+R 输入 shell:startup
2. 将 start_server.bat 的快捷方式放入此文件夹
```

### 技巧4：修改端口
如果5000端口被占用，编辑 `server.py` 最后一行：
```python
app.run(host='0.0.0.0', port=8080, debug=False)  # 改为8080或其他端口
```

### 技巧5：查看本机IP
```cmd
# 命令提示符输入
ipconfig

# 查找 "无线局域网适配器 WLAN" 下的 "IPv4 地址"
```
