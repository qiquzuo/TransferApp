### 2. **回车键自动发送文本** ⌨️

**功能说明：**
- 在文本框中按 **Enter** 键 → 自动发送
- 无需点击"发送"按钮
- 提高输入效率

**技术实现：**
```javascript
textInput.addEventListener('keydown', (e) => {
    if (e.key === 'Enter' && !e.shiftKey) {
        e.preventDefault(); // 阻止默认换行
        sendText(); // 发送文本
    }
});
```

**使用场景：**
```
输入文本："Hello World"
    ↓
按 Enter 键
    ↓
自动发送 ✅
    ↓
显示："✅ 发送成功"
```

// Shift+Enter：允许默认换行行为，不做任何处理

## 📊 性能优化

### 速率限制调整

为了支持0.1秒刷新，提高了API速率限制：

**之前：**
```python
default_limits=["200 per day", "50 per hour"]
```

**现在：**
```python
default_limits=["1000 per day", "200 per hour"]

# /api/history 接口单独设置
@limiter.limit("600 per minute")  # 每分钟600次
```

**计算：**
- 0.1秒刷新 = 每秒10次 = 每分钟600次
- 限制设置为600次/分钟，刚好满足需求

- `2000` - 2秒（省电）

### 修改速率限制

**位置：** `server.py`

**全局限制（第25行）：**
```python
default_limits=["1000 per day", "200 per hour"]
```

**历史接口限制（第289行）：**
```python
@limiter.limit("600 per minute")
```

**计算公式：**
```
刷新间隔（秒）= 60 / 每分钟限制次数

例如：
- 600次/分钟 = 0.1秒
- 60次/分钟 = 1秒
- 30次/分钟 = 2秒
```
