# 💜 欢迎使用局域网文件传输助手！

## 🎉 恭喜！你已经获得了一个完整的跨平台文件传输解决方案

---

## ✨ 这是什么？

这是一个**轻量级、美观、易用**的文件传输工具，让你可以在：
- 🖥️ **Windows电脑** 
- 📱 **Android手机**

之间通过**同一WiFi网络**快速传输：
- 📸 照片和图片
- 📄 各种文件（文档、视频、音频等）
- 📝 文本内容
- 🔗 网页链接

**无需互联网、无需蓝牙配对、无需扫码连接！**

---

## 🚀 立即开始（3步搞定）

### 第1步：启动Windows服务器
```bash
双击运行：start_server.bat
```
你会看到类似这样的输出：
```
📍 服务器地址: http://192.168.1.100:5000
```

### 第2步：手机连接
**方法A（推荐）：** 手机浏览器直接访问显示的地址  
**方法B：** 编译并安装Android App

### 第3步：开始传输
- 拖拽文件到网页
- 或点击按钮选择文件
- 或输入文本发送

**就这么简单！** 🎊

## 📞 需要帮助？

### 常见问题速查

**Q: 如何启动服务器？**  
A: 双击 `start_server.bat`

**Q: 手机如何连接？**  
A: 浏览器访问显示的IP地址

**Q: 连接失败怎么办？**  
A: 检查是否在同一WiFi，查看 [TESTING.md](TESTING.md)

**Q: 支持哪些文件类型？**  
A: 图片、文档、视频、音频等15+种格式

**Q: 文件大小有限制吗？**  
A: 默认100MB，可修改配置

更多问题 → [TESTING.md](TESTING.md) 常见问题章节
